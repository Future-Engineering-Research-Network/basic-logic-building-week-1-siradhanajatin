## 🎯 Your Tasks (3 Questions)

### **Question 1** (Mandatory - 35 points)
Write a Python program to perform **basic list operations WITHOUT using ANY built-in list functions**:

**Required operations** (implement manually):
- Find length of list
- Add element at end (append)
- Insert element at specific position
- Remove element at specific position  
- Search for element (return index)
- Find maximum and minimum elements

### **Question 2** (Mandatory - 35 points)
Write a Python program to find the **nth Fibonacci number** OR generate **Fibonacci series up to n terms**:
- Accept `n` from user input
- Handle edge cases (n=0, n=1, negative n)
- Show both iterative and recursive approaches

### **Question 3** (Bonus - 30 points) 
**CREATE YOUR OWN ORIGINAL PROBLEM** at similar difficulty:
- Must be solvable in Python
- **Cannot copy from internet/books/friends/AI**
- Same level as Q1 & Q2
- **Best 3 creative questions**: +10, +7, +5 bonus points!
